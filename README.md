# Customer Retention & Churn Analysis

## Objective
To analyze customer churn patterns and identify factors affecting retention.

## Tools Used
- MS Excel

## Key Insights
- Quarterly subscription has the highest churn
- Monthly plan also shows significant churn
- Yearly plan has the lowest churn
- Around 30% customers have churned

## Recommendations
- Promote yearly subscription plans
- Provide discounts for long-term users
- Improve service for quarterly users

- Excel dataset with analysis
- Report document
